Для сайта по дезинфекции m-ed.ru

Как сделать и проверить корректность https://texterra.ru/blog/chek-list-optimizatsii-sayta-ili-100-prichin-ne-khoronit-seo.html

1. robots.txt
2. Корректный sitemap.xml
3. Нет технических дублей страниц
4. На сайте нет битых ссылок
5. Установить сертификат https
6. Установить генератор ЧПУ УРЛов для новых страниц
7. Для существующих станиц необходимо создать УРЛы с ЧПУ. То есть не m-ed.ru/tarakani, a m-ed.ru/unochtojenie-tarakanov
Ключевые слова для создания ЧПУ УРЛов у страниц, которые уже есть в дизайне (для всех пунктов меню, которые надо сделать по дизайну) я предоставлю.
8. В URL не используются заглавные буквы
9. Нет ошибок сервера
10. В верстке нет ошибок
11. Внедрена микроразметка
12. Внедрена Open Graph разметка
13. Сайт быстро загружается
14. мобильная версия. есть дизайн
15. Сайт корректно отображается в разных браузерах
16. Сервер выдерживает нагрузку
17. На сайте нет вирусов и вредоносного кода
18. В индексе нет копий сайта
19. Настроить цель в метрике - страница Контакты.
20. Должна быть возможность в админке для каждой страницы прописать title, description, keywords
21. На страницах в админке должен быть визуальный редактор с большим кол-вом функций
22. На сайте реализованы «хлебные крошки»
23. Должна быть возможность прописывать АЛЬТы к изображениям при добавлени картинок
24. В разделе Блог должны быть картинки- превью у статей, размещенных там.
25. Добавьте кнопки шеринга в социальные сети и призыв/просьбу поделиться с друзьями (в дизайне этого нет)
26. В вебмастере указать регион Москва.
27. Добавить сайт в «Google Мой Бизнес»
28. Установить фавикон на сайте (он есть в дизайне на главной странице)
29. поставить счетчик метрики
30. добавить сайт в яндекс вебмастер
31. добавить сайт в гугл аналитикс
32. добавитьорганизация в яндекс справочник (возможно, сам добавлю)
